    Methodology in three sentences

    For every tracked model, COI computes an effective $/task by combining input/output token counts at a 2000-in/600-out reference mix with the provider's published rates, scaled by a tokenizer factor that captures changes in tokens consumed for identical text. The Frontier Floor is the cheapest frontier-tier model by that metric; the Implied Margin compares that price to SemiAnalysis cost-to-serve estimates; the Capacity Overhang is hyperscaler compute capex divided by reported token demand growth. Three signals must flip before the verdict reads subsidy ending — see the forecast page for the rolling regression that drives the projection.

    How fresh is this data?

    Daily refresh: a GitHub Actions workflow hits openrouter.ai/api/v1/models at 05:17 UTC and writes any new observations to data/prices.json. The dashboard rebuilds from that file on every load.

    Verification cycle: Anthropic, Google, DeepSeek, xAI publish prices on public pages cited per-observation. OpenAI and Mistral pages block automated fetch; OpenRouter's pass-through rate is the primary source until a contributor adds a verified scraper for those two.
